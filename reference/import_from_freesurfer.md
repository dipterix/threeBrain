# Import from \`FreeSurfer\` and create \`JSON\` cache for 3D viewer

Import from \`FreeSurfer\` and create \`JSON\` cache for 3D viewer

## Usage

``` r
import_from_freesurfer(fs_path, subject_name, quiet = FALSE)
```

## Arguments

- fs_path:

  \`FreeSurfer\` subject directory

- subject_name:

  subject code

- quiet:

  whether to suppress message or not

## Value

None.
